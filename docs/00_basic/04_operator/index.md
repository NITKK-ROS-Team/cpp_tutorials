# 演算子

コンピュータは`計算機`と呼ばれる通り、無数の演算を行うことで複雑な処理を実現しています。

プログラム上では演算子と呼ばれる記号を用いて、演算を行います。

<br>

<br>

実際に簡単な演算を行いましょう。

```cpp
#include <cstdio>

int main()
{
    int a = 1;
    int b = 2;
    int c = a + b;
    printf("c = %d\n", c);

    return 0;
}
```

出力結果は以下のようになります。

```bash
c = 3
```

<br>

演算代入を行うと、以下のようになります。

```cpp
#include <cstdio>

int main()
{
    int a = 1;
    int b = 2;
    a += b;
    printf("a = %d\n", a);

    return 0;
}
```

この処理における`a += b`は、`a = a + b`と同じ意味になります。

出力結果は以下のようになります。

```bash
a = 3
```

ほとんどの場合は、四則演算と比較演算子を覚えておけば大丈夫です。
